PORTFOLIO BOTU
Bu discord botu gerekl komutlar yazıldığında kişinin listeye tanımladığı yeteneklerini ve geçmiş projelerini listeler.örneğin başına ünlem gelecek şekilde yani !projects yazdığımızda eğer daha önce kaydedilmiş bir proje yoksa size önce proje kaydedin der ve !new_projects bu komutla yeni projenizi kaydedersiniz. Daha sonra !projects dediğiğinizde kaydettiğiniz projeyi gösterir.
